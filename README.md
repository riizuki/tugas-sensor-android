# Tugas Sensor Android

Aplikasi Android yang memanfaatkan perangkat sensor pada Android.

## Fitur

- **Kamera** - Mengambil gambar menggunakan kamera perangkat
- **GPS** - Menampilkan lokasi (Latitude & Longitude)
- **Sensor Suhu** - Mengukur suhu lingkungan (Ambient Temperature)

## Tools

- Android Studio
- Java
- Min SDK 24
